Web application to record time logs

Technologies :

  • React js
  • Node js
  • MySql

Project configuration :

Node js (Folder server)

  • Clone project git : https://github.com/mdevhd/allactivity_reactjs_nodejs_mysql.git
  • Open terminal : >cd server & npm install
  • Duplicate server/.env.dist file & rename to .env to define MySQL database configuration
  • Run node server node index.js (server is running on port 3001)

React js (Folder client)

  • Open terminal : >cd client & npm install
  • Start Reactjs Application by npm start
